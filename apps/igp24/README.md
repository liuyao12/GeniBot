# IGP24 GCTS

Placeholder for a browser app exploring geometrically constrained tree search for the degree-24 inverse Galois problem.

## Intended scope

- represent partial arithmetic constructions as reusable constraint states;
- visualize the residual completion geometry and learned conflicts;
- run construction templates and partial viability checks;
- export verified degree-24 polynomials in the competition submission format.

The app is currently a lightweight static scaffold. Its first implementation lane will focus on imprimitive constructions, beginning with elementary-abelian extension layers.
